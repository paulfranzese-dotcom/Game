# Multiplication Quest

Kid-friendly multiplication practice app (0-12) built with React + Vite + TypeScript.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Build check:
   ```bash
   npm run build
   ```
3. Start development server:
   ```bash
   npm run dev
   ```

## Included features
- Player profile with persistent local storage.
- Per-player save files by name, with Switch Player / Start Over flow.
- Adventure, Practice, Speed Round, Boss Battle, and Mistake Review modes.
- Spaced repetition weighting for missed facts.
- Immediate answer feedback with explanation.
- XP, coins, badges, and level progression.
- Reward pop-ups (+XP, +Coins, streak, badge, session complete).
- Sound effects (correct, wrong, coins/XP, badge unlock, session complete) with mute toggle.
- Progress map and parent-friendly dashboard summary.
- Kid-friendly animations for correct, wrong, streak, badge, and session completion.
- Accessible large controls and keyboard answer submission.
